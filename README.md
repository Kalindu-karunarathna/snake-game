A classic Snake game built using Python's Turtle module. The snake moves around the screen, eats food to grow longer, and the game ends if it collides with the wall or itself. 

---

## Features

- Smooth snake movement using Python Turtle
- Food randomly spawns on the screen
- Collision detection with walls and the snake's own tail
- Score tracking displayed at the top of the screen
- Increase the speed of the snake according to score
- Sound effects for eating food and game over
